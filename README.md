# Password Manager CLI

Gestor de contraseñas local y seguro implementado en Bash Script. El sistema almacena los datos cifrados mediante criptografía simétrica empleando OpenSSL y renderiza la información de forma dinámica utilizando tablas con caracteres de caja Unicode (UTF-8).

## Requisitos del sistema

- Sistema operativo compatible con POSIX (GNU/Linux, macOS, Windows Subsystem for Linux).
- Bash en versión 4.0 o superior.
- Utilidad openssl disponible en el PATH del sistema.

## Instalación

1. Clone el repositorio en su máquina local:
   ```bash
   git clone git@github.com:Guille-Lavado/password-manager-cli.git
   cd password-manager-cli
   ```
